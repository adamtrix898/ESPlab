Sensor de Temperatura con Arduino

Mide temperatura con el sensor LM35 y muestra por el monitor serial.

Materiales

  • 1 Arduino UNO
  • 1 Sensor LM35
  • Protoboard y cables

Conexiones

Salida del LM35 al pin A0, VCC a 5V, GND a GND.

Código


void setup() {
  Serial.begin(9600);
}

void loop() {
  int valor = analogRead(A0);
  float voltaje = valor * 5.0 / 1023.0;
  float tempC = voltaje * 100;
  Serial.print("Temperatura: ");
  Serial.print(tempC);
  Serial.println(" °C");
  delay(1000);
}
