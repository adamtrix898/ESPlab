Robot sigue líneas

Un robot autónomo que sigue trayectorias negras usando sensores infrarrojos.

Materiales

  • 2 Motores DC
  • 1 Placa Arduino UNO
  • 2 Sensor Infrarrojo TCRT5000
  • Driver L298N
  • Ruedas + Chasis

Conexiones

Conecta los sensores IR a los pines digitales del Arduino. Los motores al driver y el driver al Arduino.

Código


void setup() {
  pinMode(2, INPUT); // sensor izquierda
  pinMode(3, INPUT); // sensor derecha
  pinMode(9, OUTPUT); // motor A
  pinMode(10, OUTPUT); // motor B
}

void loop() {
  int izquierda = digitalRead(2);
  int derecha = digitalRead(3);

  if (izquierda == LOW && derecha == LOW) {
    digitalWrite(9, HIGH);
    digitalWrite(10, HIGH);
  } else if (izquierda == LOW) {
    digitalWrite(9, HIGH);
    digitalWrite(10, LOW);
  } else if (derecha == LOW) {
    digitalWrite(9, LOW);
    digitalWrite(10, HIGH);
  } else {
    digitalWrite(9, LOW);
    digitalWrite(10, LOW);
  }
}
