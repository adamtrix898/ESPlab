Control de LED con Arduino

Enciende y apaga un LED con un botón.

Materiales

  • 1 Arduino UNO
  • 1 LED
  • 1 Resistencia de 220Ω
  • 1 Pulsador

Conexiones

LED en el pin 13 con resistencia. Pulsador entre pin 2 y GND.

Código


const int ledPin = 13;
const int boton = 2;
int estado = 0;

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(boton, INPUT);
}

void loop() {
  estado = digitalRead(boton);
  if (estado == HIGH) {
    digitalWrite(ledPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
  }
}
